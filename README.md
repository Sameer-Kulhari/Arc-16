# Arc-16

Arc-16 is a macro-pad with 16 keys,oled and rgb for aesthetics and look ,with a custom cool looking 3d case and working on kmk firmware .

## Features :
- 16 keys 
- Oled (128*32)
- Rgb under keys 
- kmk firmware
- 3 layers for rgb and functions control keys
